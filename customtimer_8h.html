uint8_t prepare_countdowns ( uint16_t  n_cds,
float *  seconds,
T_CALLBACK *  callbacks 
)

Prepare a number of countdowns.

Overwrite all existing countdowns with the parameters. This function will fail, either when a countdown is currently running or when the desired number of countdowns exceeds MAX_COUNTDOWNS.

Returns
1 on success, 0 on failure.
Parameters
n_cdsNumber of desired countdowns
secondsArray of length <n_cds>, containing the time for each countdown
callbacksArray of length <n_cds>, containing pointers to the callbacks for each countdown
